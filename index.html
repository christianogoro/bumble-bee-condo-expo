Google Sheets Sync

To sync leads live to Google Sheets, you need a Google Apps Script web app URL.

Quick setup:
1. Create a Google Sheet with headers: Timestamp, First Name, Last Name, Role, Community, Mgmt Company, Email, Phone, Units, Property Type, Products, Scope, Stage, Timeline, Approver, Notes, Promo
2. Go to Extensions → Apps Script
3. Paste the script (tap "Copy Script" below)
4. Deploy → New Deployment → Web App → Anyone can access
5. Paste the deployment URL below
